<img width="1165" height="851" alt="image" src="https://github.com/user-attachments/assets/fabc5dd8-94b5-4d80-93d4-380bb276b15e" /># Rewynd Packaging – Sticker Generator

A web-based Sticker Generator developed for Rewynd Packaging to automate packaging label creation.

This tool generates structured, print-ready stickers based on packaging details entered by the user.

---

## 🖥️ Application Overview

The system allows users to input:

- Serial Number
- Size (e.g., 95x102cm)
- GSM (e.g., 250gsm)
- Packaging Details
- Number of Bundles (Stickers)

After clicking **Generate Stickers**, the system dynamically creates formatted sticker outputs ready for printing.

---

## 🎯 Purpose

This tool reduces manual errors in packaging labeling and improves operational efficiency during dispatch.

It is designed specifically for packaging and box manufacturing workflows.

---

- Clean and minimal UI

- Bundle-based sticker calculation

## 📊 Data Fields Used

| Field | Description |
|-------|------------|
| Serial No | Product batch reference |
| Size | Sheet dimension |
| GSM | Paper thickness |
| Packaging Details | Quantity structure |
| No. of Bundles | Total stickers to generate |
